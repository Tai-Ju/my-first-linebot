Tech Stack
技術棧
🐍
Python · Flask
輕量級 Web 框架，處理 LINE Webhook 事件，路由設計簡潔清晰。
FlaskPython 3dotenv
💬
LINE Messaging API v3
支援文字、貼圖、地理位置、圖片等訊息類型，使用官方 Python SDK。
linebot.v3Webhook
🧠
OpenAI GPT-4o
文字問答使用 Chat Completion，圖片辨識使用 Vision API（base64 圖片輸入）。
gpt-4oVisionbase64

Render
免費雲端部署平台，自動提供 HTTPS 端點作為 LINE Webhook URL。環境變數安全管理。
ServerlessHTTPS

FAQ 模組

dict模組化
